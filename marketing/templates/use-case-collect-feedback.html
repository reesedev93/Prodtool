From Intercom

When you connect Savio to Intercom, you'll give Savio the name of an Intercom tag that you want Savio to pull in. When your teammates add that tag to an Intercom message, Savio will pull that message in. Tagging doesn't work in Intercom's mobile apps, but with Savio there's a way to push customer feedback from Intercom's mobile apps. Learn more about tracking feature requests sent to Intercom →

From Help Scout

When you connect Savio to Help Scout, you'll give Savio the name of an Help Scout tag. When your support team adds that tag to a Help Scout conversation, Savio will pull that message in as new feedback. Learn more about tracking feature requests sent to Help Scout →

With a Chrome Extension

Your probably have feedback that lives outside of Intercom or Slack (Google Docs, an NPS tool, and a CRM come to mind). It's problematic to get that feedback into a single place. But with the Savio Chrome Extension, you can push feedback to Savio from any web-based tool. Learn more about tracking feature requests with the Savio Chrome Extension →

From Slack

When you connect Savio to Slack, Savio can listen to the channel where your team regularly post feedback. Savio will ask the person who shared it if they want to to push it to Savio. Savio also provides a way to push any public message in Slack to Savio. Learn more about tracking feature requests shared in Slack →

From any tool via Zapier

Track feature requests in Savio from any tool with our Zapier integration. Customers have used Zapier to send feature requests to Savio from tools like Survey Monkey, NPS tools, and Typeform. Learn more about tracking feature requests via Zapier →

From Your Email Client

If you're like us, you get tons of valuable feedback via email. With Savio, you can email that feedback from your inbox into your Vault. Learn more about tracking feature requests via email →

With the Savio API

Do you collect qualitative feedback in your app? Want to keep it in the same place as your other feedback? Send it to Savio with our API. Learn more about tracking feature requests with the Savio API →

Directly in Savio

As you might expect, you can enter customer feedback directly into Savio after you log in.
