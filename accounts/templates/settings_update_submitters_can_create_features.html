Can Submitters Create Feature Requests?

If enabled, your teammates with the "Submitter" role can create Feature Requests from the Chrome Extension. If disabled, they can only select from existing Feature Requests and can't create new ones.

It's useful to disable this if you want your product team to control the list of features in Savio.
