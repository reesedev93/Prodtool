To Triage this Feedback

  1. Decide whether you'll act on this Feedback someday. If you won't, delete it
  2. Attach the Feedback to a Feature Request. This "votes" up the Feature Request
  3. Mark it as Triaged. You can still find it by clicking "All Feedback" in the sidebar
